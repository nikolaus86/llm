УСТАНОВКА И ЗАПУСК:

1. Установите зависимости:
pip install -r requirements.txt

2. Для локального развертывания моделей:
- LM Studio: https://lmstudio.ai/
- Ollama: https://ollama.ai/

3. Рекомендуемые модели для начала:
- Qwen-2.5-1.5B (наименьшие требования)
- Llama-3.2-3B (оптимальное качество)

4. Запуск демонстрации:
python demo.py

ТРЕБОВАНИЯ К ЖЕЛЕЗУ:
- Минимум 8GB RAM
- 4GB VRAM для моделей ~3B параметров
- 16GB RAM для моделей ~7B параметров
